🎯 Gráfico V/E/D — Forma Recente
Bloco 1 de 6 · 08:00–10:00 · 2 horas · Claude Sonnet
OBJETIVO
O botão GRÁFICO da seção "Forma Recente" precisa exibir um gráfico de barras mostrando V/E/D (Vitórias / Empates / Derrotas) dos últimos 5 jogos do clube selecionado.
PASSOS
01Abrir GitHub Codespace do placar-analitico
02Localizar o bloco da Forma Recente no JS (provavelmente renderFormaRecente ou similar)
03Verificar se os dados V/E/D já existem no JSON ou precisam ser calculados a partir dos resultados
04Criar canvas Chart.js ou elemento SVG para o gráfico de barras empilhadas/agrupadas
05Cores: Verde (V), Amarelo (E), Vermelho (D) — consistentes com o resto do dashboard
06Ligar ao toggle: botão GRÁFICO mostra o chart, botão DETALHES mostra a lista atual
07Testar com 3 clubes diferentes para garantir que os dados carregam corretamente
08Commit: "feat: forma recente — gráfico V/E/D implementado"
